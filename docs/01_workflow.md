# Workflow (propuesto)

## Estructura de trabajo
- Upright (Regular + Bold): `sources/upright/`
- Italic (Italic + BoldItalic): `sources/italic/`

## Exportación
- Builds de prueba: `builds/otf/proof/`
  - Sugerencia de nombre: `LEDS_Text.otf` o `YYYYMMDD`.
- Release: `builds/otf/release/`
  - Solo archivos finales aprobados.

## Registro de cambios
- Cambios importantes (espaciado, kerning, sets de caracteres, curvas): `CHANGELOG.md`.
- Acuerdos de reunión: `docs/reuniones/notas.md`.
